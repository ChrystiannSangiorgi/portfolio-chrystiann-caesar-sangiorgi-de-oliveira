# Portfólio de IA Generativa e Engenharia de Software

Bem-vindo ao meu repositório acadêmico! 

Este espaço consolida projetos práticos e laboratórios focados em Inteligência Artificial, Engenharia de Prompt, Desenvolvimento Assistido por IA (VibeCoding) e concepção de Produtos Mínimos Viáveis (MVPs). 

## 📂 Estrutura de Projetos

* **[A Corrida do Prompt](./projeto-a-corrida-do-prompt):** Exercícios de iteração e refinamento de prompts com limite de tentativas.
* **[Batalha de Modelos e Engenharia de Prompt](./projeto-batalha-de-modelos-e-engenharia-de-prompt):** Análise comparativa de LLMs utilizando estruturação em XML.
* **[Do Clone ao Produto Mínimo Viável](./projeto-do-clone-ao-produto-minimo-viavel):** Evolução de um clone para um produto autoral com diferenciais competitivos.
* **[Engenharia Reversa](./projeto-engenharia-reversa):** Recriação de aplicações web a partir da interface visual usando IA como copiloto.
* **[Laboratório de Classificação Visual](./projeto-laboratorio-de-classificaçao-visual):** Testes práticos com IA para reconhecimento de padrões (Documentação em andamento).

Sou estudante do 2º semestre de Ciência da Computação, em transição de carreira para a área de Tecnologia, com objetivo de atuar em Arquitetura Cloud e Engenharia de Dados.
