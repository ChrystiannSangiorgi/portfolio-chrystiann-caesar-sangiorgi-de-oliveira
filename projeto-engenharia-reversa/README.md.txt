# Engenharia Reversa (VibeCoding)

## 📌 Sobre o Projeto
Neste projeto, assumi o papel de Desenvolvedor Full-Stack assistido por IA[cite: 8]. A missão foi realizar a engenharia reversa de um webapp (como o QR-Style-Max), reconstruindo a aplicação inteira a partir da observação exclusiva de sua interface visual, sem nenhum acesso ao código-fonte original[cite: 5, 8].

## ⚙️ Metodologia de Desenvolvimento
* **VibeCoding no Google AI Studio:** Utilizei o modelo Gemini, configurando System Instructions para ditar o comportamento esperado, estrutura de arquivos (HTML/CSS/JS) e lógica de negócio[cite: 8].
* **Validação:** Geração e execução do código em ambiente de teste com ajuste de prompts até igualar a funcionalidade da aplicação de referência[cite: 8].

## 🧠 A Evolução do Engenheiro de Software
* **De Digitador para Auditor:** A IA remove a fricção sintática (como esquecer pontuações), o que exige que o desenvolvedor passe a focar em como os sistemas se conectam[cite: 7].
* **Visão Sistêmica:** O entendimento de Estados Globais, Ciclos de Vida e Fluxo de Dados torna-se muito mais importante do que a simples memorização de sintaxe[cite: 7]. O papel do júnior passa a ser o de um arquiteto de soluções[cite: 7].
