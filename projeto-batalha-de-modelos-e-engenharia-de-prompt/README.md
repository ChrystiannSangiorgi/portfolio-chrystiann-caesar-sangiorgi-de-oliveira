# Batalha de Modelos & Engenharia de Prompt (XML)

## 📌 Sobre o Projeto
Este laboratório consistiu em uma análise crítica comparativa entre diferentes Large Language Models (LLMs) disponíveis no mercado (ChatGPT, Gemini, Claude, Qwen, DeepSeek, Grok e Maritaca)[cite: 3, 4]. O objetivo foi avaliar a precisão técnica e a conformidade de cada modelo em relação a um conjunto estrito de instruções estruturadas em XML[cite: 4].

* **Equipe:** Victoria Cerqueira Viana, Chrystiann Caesar Sangiorgi de Oliveira, Gabriel Paiva Da Silva Souza[cite: 3].

## 🛠️ O Experimento
Foi desenvolvido um Prompt Estruturado em XML exigindo a criação de uma página HTML5 Single Page, com CSS interno, voltada para uma "Busca Inteligente de Estágios"[cite: 3]. As diretrizes exigiam paleta de cores específica, menu de âncoras e design minimalista[cite: 3].

## 📊 Resultados e Análise Crítica
* **Precisão e Compreensão:** O modelo DeepSeek demonstrou a melhor compreensão da estrutura XML, entregando exatamente o solicitado sem erros, sendo a escolha ideal para códigos complexos[cite: 3].
* **Verbosidade (Consumo de Tokens):** Houve grande discrepância na geração de código[cite: 3, 4]. O ChatGPT gerou o resultado gastando entre 900 a 1100 tokens, enquanto o Claude consumiu até 4.800 tokens para a mesma tarefa[cite: 3].
