# Laboratório de Classificação Visual e Ética em IA

## 📌 Sobre o Projeto
Neste laboratório, o objetivo foi utilizar o Google Teachable Machine para treinar um modelo de classificação de imagens de forma deliberadamente enviesada[cite: 11]. A atividade focou em compreender na prática como a seleção restrita de dados afeta o comportamento de uma Inteligência Artificial e debater suas consequências éticas[cite: 10, 11].

## ⚙️ O Experimento Prático
* Foi desenvolvido um modelo de visão computacional para categorizar mochilas em duas classes baseadas em estereótipos: "playboy" e "quebrada"[cite: 9, 10].
* O modelo foi treinado intencionalmente com pouquíssimos dados para gerar um "Dataset Enviesado", forçando o algoritmo a distorcer a maneira de categorizar as imagens e demonstrar falhas de lógica no discernimento[cite: 10, 11].
* **Link do Modelo Treinado:** [Teachable Machine - Model n1nESMmU6](https://teachablemachine.withgoogle.com/models/n1nESMmU6/)[cite: 10].

## ⚖️ Memorial de Impacto e Ética
A partir do experimento técnico, foram analisados os impactos reais do viés algorítmico:
* **Mecanismo do Viés:** A falta de dados adequados para o treinamento, aliada à ausência de uma confirmação humana ("Human-in-the-loop"), corrompe a lógica do algoritmo e gera uma visão distorcida da realidade[cite: 10, 11].
* **Consequência Social:** Sistemas que utilizam classificações como "playboy" ou "quebrada" marginalizam indivíduos, podendo fazer com que se sintam menosprezados devido à sua condição social e de classe[cite: 10, 11].
